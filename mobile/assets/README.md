# Assets

Place the following asset files in this directory:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1284x2778 recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `favicon.png` - Web favicon (48x48)

All icons should use the green color (#16a34a) and feature the Consumemate logo.
