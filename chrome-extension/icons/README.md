# Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

Icons should be PNG format with transparent background and feature the Consumemate logo (a green "C" or book icon).

For development, you can generate placeholder icons using an online tool or design software.
